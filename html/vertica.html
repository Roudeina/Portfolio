VERTICA

Thesis Project

Vertica is a social media application, in which plants lovers are reunited to exchange advice, knowledge or suggestions. The application include a blog section that provide articles written by experts to help users take care of their gardens. And ‘Contact expert’ section that regroup all our verified experts to facilitate to the user finding all information that he can need.

Technologies used

React : library for building user interfaces combined with Redux , Bootstrap and MaterialUi.

NodeJS : back-end JavaScript runtime environment.

MongoDb: noSQL document database for storage.

Express: Node.js web application framework.

Firebase: a Google’s application development platform .

Cloudinary: image cloud storage.
