MY UNSPLASH

My Unsplash is a challenge to fulfill user stories below:

User story: I can see a list of photos in the masonry layout that I have added

User story: I can add a new photo to the list - the new photo should be on top of the list

User story: I can search for photos by label

User story: When I hover a photo, I can see a label and a delete button

User story: I can delete images

Technologies used

Angular : framework for building client applications using HTML and TypeScript combined with AngularMaterial and Bootstrap.

NodeJS : back-end JavaScript runtime environment.

MongoDb: noSQL document database for storage.

Express: Node.js web application framework.

Cloudinary: image cloud storage.
